## momoclo ustream archive viewer ##

### WebSite ###

http://momoclo.no.de/

### Setup ###

    $ git clone git://github.com/sugyan/momocloop.git
    $ cd momocloop
    $ npm install
    $ npm start

### Requirement ###

- [node.js](http://nodejs.org/)
- [MongoDb](http://www.mongodb.org/)
- [Redis](http://redis.io/)
