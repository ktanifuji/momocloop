## momoclo ustream archive viewer ##

### WebSite ###

http://momocloop.sugyan.com/

### Setup ###

    $ git clone git://github.com/sugyan/momocloop.git
    $ cd momocloop
    $ git submodule update --init
    $ npm install
    $ npm start

### Requirement ###

- [node.js](http://nodejs.org/)
- [Redis](http://redis.io/)
